# Learning Resources

Curated resources by phase. Free unless marked [PAID].

---

## Phase 1 — HTML & CSS

### Primary (pick one to follow)
- **The Odin Project** — https://www.theodinproject.com/paths/foundations
  - Best free, project-based curriculum. Highly recommended as your main guide.
- **freeCodeCamp Responsive Web Design** — https://www.freecodecamp.org/learn/2022/responsive-web-design/
  - Good for structured exercises with immediate feedback.

### Reference (bookmark these, use when stuck)
- MDN HTML docs — https://developer.mozilla.org/en-US/docs/Web/HTML
- MDN CSS docs — https://developer.mozilla.org/en-US/docs/Web/CSS
- CSS Tricks Flexbox Guide — https://css-tricks.com/snippets/css/a-guide-to-flexbox/
- CSS Tricks Grid Guide — https://css-tricks.com/snippets/css/complete-guide-grid/

### Practice
- Flexbox Froggy — https://flexboxfroggy.com/ (learn flexbox by playing)
- Grid Garden — https://cssgridgarden.com/ (learn grid by playing)

## Phase 2 — JavaScript + REST APIs

### Primary
- **JavaScript.info** — https://javascript.info/
  - The best free JavaScript textbook. Read it, don't skip it.
- **freeCodeCamp JavaScript Algorithms** — https://www.freecodecamp.org/learn/javascript-algorithms-and-data-structures/

### Reference
- MDN JavaScript docs — https://developer.mozilla.org/en-US/docs/Web/JavaScript

### Practice
- Exercism (JavaScript track) — https://exercism.org/tracks/javascript
  - Small coding exercises with mentor feedback. Use from week 2 onwards.

### REST APIs & HTTP (study these during weeks 4-5 of Phase 2)
- **What is a REST API?** — https://www.restapitutorial.com/
  - Read the "Learn REST" section. Short and clear.
- **HTTP Crash Course** — search "HTTP Crash Course Traversy Media" on YouTube
  - 30 min video, explains GET/POST/PUT/DELETE, status codes, headers
- **JSONPlaceholder** — https://jsonplaceholder.typicode.com/
  - A fake REST API for practice. Use it to try GET/POST requests before using real APIs.
- **Public APIs list** — https://github.com/public-apis/public-apis
  - A huge list of free public APIs to practice with (weather, movies, jokes, etc.)
- Hoppscotch (free API tester, like Postman) — https://hoppscotch.io/
  - Use this to test API calls visually before writing code

## Phase 6 — Next.js + Backend + Databases

### Next.js
- **Official Next.js Tutorial** — https://nextjs.org/learn
  - Work through this top to bottom. Excellent quality.
- Next.js Docs — https://nextjs.org/docs

### SQL & Databases (learn before touching Supabase)
- **SQL Murder Mystery** — https://mystery.knightlab.com/
  - Learn SQL by solving a crime. Fun and practical. Do this first.
- **SQLiteOnline** — https://sqliteonline.com/
  - Practice SQL in the browser with no setup needed.
- W3Schools SQL — https://www.w3schools.com/sql/
  - Quick reference for SELECT, INSERT, UPDATE, DELETE, JOIN

### Supabase
- **Supabase official docs** — https://supabase.com/docs
  - Start with "Getting Started" → then "Database" → then "Auth"
- **Supabase + Next.js quickstart** — https://supabase.com/docs/guides/getting-started/quickstarts/nextjs
  - Follow this when you start your first fullstack project

### Authentication Concepts
- What is JWT? — search "JWT explained in 5 minutes" on YouTube
- What are cookies vs localStorage for auth? — ask Claude Code to explain this when you get here
